# 🎬 Movie Recommendation System

## 📌 Description
This project is a movie recommendation system that suggests movies based on user preferences using machine learning / filtering techniques.

## 🚀 Features
- Search movies
- Get personalized recommendations
- Watchlist functionality
- User authentication (if added)

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js / Express
- Database: (MongoDB / etc.)
- Styling: Tailwind CSS

## ▶️ How to Run

### Frontend
cd frontend
npm install
npm start

### Backend
cd backend
npm install
npm start
